# 🚀 Binance Alpha Scanner V3

一个自动化扫描系统，用于从 Binance Alpha 页面发现早期低市值代币，并通过多数据源交叉验证筛选出真实可交易资产。

---

# 🧠 一、项目原理（非常重要）

本项目不是简单爬虫，而是一个 **三数据源过滤系统**：

### 🔍 数据流程：

1️⃣ 从 Binance Alpha 获取新币（早期信号）  
2️⃣ 用 Dexscreener 验证流动性（是否真实交易）  
3️⃣ 用 Binance Spot 验证是否已上线  

---

### 📊 为什么要这样做？

| 问题 | 解决方案 |
|------|--------|
| Alpha 有垃圾币 | Dex 过滤 |
| 有假流动性 | 流动性阈值 |
| 未上线币风险高 | Binance Spot 标记 |

---

# ⚙️ 二、功能说明

✔ 自动扫描 Alpha 页面  
✔ 筛选市值 < $1M  
✔ 过滤垃圾币 / 空气币  
✔ 验证真实流动性  
✔ 判断是否已上线 Binance  
✔ 自动写入 Google Sheet  
✔ 每天定时运行（08:00 / 20:00 北京时间）
